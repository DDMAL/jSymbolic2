jSymbolic Tutorial - jSymbolic Basics

GETTING BASIC INFORMATION ABOUT JSYMBOLIC

  • There is a menu called Information at the top-left of the jSymbolic GUI window
  • The About option brings up information about the version of jSymbolic that you have installed
  • The Help option brings up the jSymbolic manual
    • It is usually easier to read the manual in a web browser, however
  • The Tutorial option brings up this tutorial
    • It is usually easier to read the tutorial in a web browser, however

THE INFORMATION PANELS

  • The Processing Information Panel, on the middle left of the jSymbolic GUI, is a space where:
    • Startup stats about jSymbolic are posted
    • Updates are posted when files are added or removed from the list of files to process
    • Processing updates are posted during feature extraction
    • Information about configuration files is posted
  • Take a look at what is shown in the Processing Information Panel now:
    • A note that default configurations were loaded at startup from the "jSymbolicDefaultConfigs.txt" file (more on config files later)
    • A breakdown of all the features chosen to be extracted by default, as well as all the features available in general
  • The Error Reports Panel, on the middle right of the jSymbolic GUI, is a space where:
    • Any problems encountered during feature extraction processing are posted
      • Such problems typically relate to invalid files
      • Feature extraction continues without interruption even if a problem occurs
    • If everything goes well you will not be spending much time with this panel!
